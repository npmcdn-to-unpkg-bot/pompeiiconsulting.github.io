Information Security and Controls

COBIT Framework · information security · data security · information controls · information technology governance · 2016-07-11

Using the COBIT framework we will explore how a wide range of information security, data security, and information controls are designed and implemented. Further, we will look at how this leads to effective information technology governance.
This will help us develop an intuitive understanding of information security and information technology governance. We will then asses the effectiveness of different forms of information security and varying iterations of organizational information security program designs.
